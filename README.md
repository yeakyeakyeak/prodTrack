# ProdTrack - Personal Productivity & Finance Manager

**ProdTrack** is a modern, lightweight web application for managing daily tasks, tracking habits, and monitoring personal finances. Built with pure HTML, CSS, and JavaScript, it provides a clean, intuitive interface for boosting productivity and financial awareness.

The application features two integrated modules: a Task & Habit Tracker and a Budget Manager, both designed to work seamlessly together while storing data locally in the browser.
